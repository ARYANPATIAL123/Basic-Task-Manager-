# Backend - TaskManagerAPI

Requirements:
- .NET 8 SDK

How to run (macOS):
1. Open terminal.
2. cd backend/TaskManagerAPI
3. dotnet run
4. The API will run on https://localhost:5001 (or http://localhost:5000). Endpoints:
   - GET  /api/tasks
   - POST /api/tasks
   - PUT  /api/tasks/{id}
   - DELETE /api/tasks/{id}
